Examples

Call an alert.

Sample:
Code:
$("#button1").undoCountdown({ 
	seconds: 3,
	onComplete: function() {
		alert("It's the final countdown.");
	}
});

Remove a table row.

Sample:
House Mascot Colors Remove
Gryffindor Lion Scarlet and Gold
Hufflepuff Badger Yellow and Black
Ravenclaw Eagle Blue and Bronze
Slytherin Serpent Green and Silver
Code:
$(".remove-row").each(function () {
	var $row = $(this).parents("tr");
	$(this).undoCountdown({
		onComplete: function() {
			$row.remove();
		}
	});
});
